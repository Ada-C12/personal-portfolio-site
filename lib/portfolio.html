Portfolio

A selection of my programming projects.

Slack terminal interface (with Samantha Collares)

This program interacts with Slack's API. Users can view workspace details and send messages from the command line (should they ever wish to do so).

Hotel reservation management tool

This is a backend program that allows the staff of a small hotel to view a list of rooms, see room availability and make reservations for a single date or a date range.

Rideshare dispatcher (with Cloudy Lopez)

A backend program for managing rideshare operations. When a dispatcher requests a ride for a customer, the program assigns a driver and updates driver, customer, and trip information.

Grocery store data management

This program parses inventory and customer information from csv files. It can perform simple queries and can basic inventory updates.
