Hello there!

I'm Julia, a newly sprouted full-stack software engineer.

Not sure what a full-stack engineer is? Neither was I until (very) recently!

Full-stack developers create tools for end users (like this fine website and other online dohickies) as well as the programs that make everything run behind the scenes. If you like metaphors, you could say full-stack developers work on the whole tree, from branches and leaves (the visible parts) to roots (the support structure).
