Portfolio

Hotel

The hotel project utlizes object-oriented programming principles to create the backend of a hotel reservation system. In this school project, I was given the freedom to design the program with the classes I felt would best satisfy the user needs/requirements while minimizing class dependencies and keeping code DRY.

Slack-CLI

The Slack-CLI project uses APIs to retrieve information from Slack and post messages via a SlackBot. The design of the program was prescribed and minimal additional code was provided. Using pair-programming techniques, a partner and I were tasked with completing this project practicing object-oriented programming and inheritance.

Grocery Store

The Grocery Store project utilizes Test Driven Development to create a grocery store program that tracks individual orders. This project utilizes class methods and reading CSV files.
